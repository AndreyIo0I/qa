<H1>Dead Link City - The Museum of Dead Links</H1>
<P>

74 different broken links feature on this page. Enter "www.DeadLinkCity.com" in your link checking
tool to scan this website.<BR>&nbsp;<BR>
The perfect score is 74 - the closer the result is to 74, the more accurate the tool is.<BR>&nbsp;<BR>
<B>
Which is the best Broken Link Checker? See the <A href="/comparison.asp">Comparison Table.</A>
</B> Scans performed in May 2016.
</P>
<P>This website contains <B>74</B> intentionally bad HTML links, in order to verify that they are detected by various
link-checking tools. The errors fall into a few broad categories:
</P>
<UL>
<LI>36 Simple HTML error status codes 4xx or 5xx</LI>
<LI>3 HTML &lt;A&gt; links to non-existent pages or domains.</LI>
<LI>1 HTML &lt;A&gt; link to a non-existent page which is denied to robots.</LI>
<LI>4 HTML &lt;A&gt; links which redirect in various ways to a non-existent page.</LI>
<LI>3 missing CSS background images - declared in inline style, external css file, and &lt;style&gt; element respectively.</LI>
<LI>27 missing files referenced by HTML elements other than &lt;A&gt; - for example &lt;ins cite="missing.html"&gt;No cite&lt;/ins&gt;</LI>
<BR>
<LI>There is one bad link (to 'disallowed/disallowed.html') which is 'disallowed' in the robots.txt file. Scanners which respect robots.txt should not report this link</LI>
</UL>

<P>
See <a href="errorlist.asp">here</a> for a list of the bad links on this page.
